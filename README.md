## 프로젝트 소개
* * *
<img src="/demo/main2.png" width="40%" height="30%" alt="game"></img>

> momentum은 구글 확장프로그램으로 로그인 정보 저장, 날씨, 시간, 체크리스트를 출력해주는 앱입니다. 

⭐️ __1. 배경이 랜덤하게 변화__
  

⭐️ __2. 로그인 정보, todo list 저장__

    처음 입력한 사용자의 정보를 저장하여 다음에 웹 브라우저를 열었을 떄 모든 정보가 저장
    
       
⭐️ __3. 현재시간, 날씨정보__
