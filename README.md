> 벌레잡기 게임은 random으로 벌레와 벌레의 개수가 정해지면 잔디에서 해당 벌레를 제한시간 안에 제거하는 게임입니다.

⭐️ __1. 배경이 랜덤하게 변화__

    각각 랜덤한 개수의 벌레들이 잔디에 추가
  

⭐️ __2. 로그인 정보, todo list 저장__

    처음 입력한 사용자의 정보를 저장하여 다음에 웹 브라우저를 열었을 떄 모든 정보가 저장
